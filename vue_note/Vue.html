插值语法

插值语法用于向HTML中渲染数据，也就是数据会被插入到指定的地方。mustache，大胡子语法。

使用示例

<div id="app">
    <h1>Hello {{name}}</h1>
</div>
<script>
    new Vue({
        el: "#app",
        data: {
            name: 'John'
        }
    });
</script>

能够插入什么东西

凡是能够得到返回值的变量、函数调用、表达式求值都可以写入到大胡子中。

大胡子还支持过滤器的写法，后面会了解到v-bind写法也支持过滤器。

{{name | slice}}

指令

Vue中的指令能够完成很多事情，比如动态绑定class和style、表单元素value的双向绑定、绑定事件、控制节点的显示与隐藏等等。

v-bind

v-bind用于绑定元素属性的值，这种绑定是单向的，也就是说，值只会从data流向元素的属性。

这个指令有以下三个方面的作用。

绑定普通属性

可以使用v-bind给普通属性绑定上值，比如v-bind:data-index="index"，也可以给key这个特殊的属性绑定上值。v-bind的语法糖写法为：:key="id"。

绑定class

Talk is cheap.

xxxxxxxxxx
1.字符串写法
    :class="mood" 绑定data中mood属性的值
    适用于：样式类名不确定，需要动态指定
2.数组写法
    :class="classArr"
    data: {
        classArr: ['class0', 'class1']
    }
    适用于：绑定的样式个数和类名都不确定
3.对象写法
    :class="classObj"
    适用于：个数确定 名字确定 但是不确定是否应用该样式
      data: {
          classObj: {
               style1: true,
               style2:  false
            }
        }

v-for

这个指令用于渲染一组数据，进行列表渲染。类似于for循环。

可以用来循环数组、对象、字符串、数字。

使用v-for渲染时，拥有同一父元素的子元素最好设置唯一的key属性，这样更改数据时更加快速，而且不会造成渲染错误。
